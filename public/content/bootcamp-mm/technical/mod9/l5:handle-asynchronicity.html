Getting Started

0.4.1 - 0.4.3: Bootcamp Spot Overview

Bootcamp Spot ဆိုတာ မင်းတို့ စာသင်ခန်းပါ။ ဒီထဲမှာ သင်ခန်းစာတွေကြည့်ရမယ်၊ Assignments တွေ တင်ရမယ်။

  • Global Navigation: ဘယ်ဘက်ခြမ်းမှာရှိပြီး Account, Dashboard, Calendar တွေကို သွားလို့ရတယ်။
  • Time Zone: အချိန်တွေ မမှားအောင် Account Settings ထဲမှာ ကိုယ့်နေရာက Time Zone ကို အရင်မှန်အောင် ပြင်ထားပါ။
  • Profile: ကိုယ့်ရဲ့ Profile Picture ကို အားလုံး မြင်ရအောင် တင်ထားပေးပါ။

0.4.4: Slack and Zoom

  • Slack: အတန်းဖော်တွေ၊ ဆရာတွေနဲ့ စကားပြောဖို့ သုံးပါတယ်။ မေးခွန်းရှိရင် AskBCS bot ကနေတစ်ဆင့် မေးလို့ရတယ်။
  • Zoom: Live Class တွေတက်ဖို့ သုံးပါတယ်။ အတန်းတက်ချိန်မှာ Camera ဖွင့်ထားဖို့နဲ့ စကားမပြောချိန် Mic ပိတ်ထားဖို့ လိုပါတယ်။

0.4.5 - 0.4.6: Equipment and Tools (လိုအပ်တဲ့ ပစ္စည်းနဲ့ Tools များ)

Hardware Requirements:

  • Laptop: အနည်းဆုံး 8 GB RAM ရှိရမယ်။
  • Internet: အမြန်နှုန်း ကောင်းရမယ်။
  • Accessories: Webcam, နားကြပ် နဲ့ Monitor တစ်လုံး အပိုရှိရင် ပိုအဆင်ပြေပါတယ်။

Install လုပ်ထားရမည့် Tools များ:

  • Google Chrome
  • Visual Studio (VS) Code
  • Git နှင့် GitHub
  • Git Bash (Windows သုံးသူများအတွက်)
