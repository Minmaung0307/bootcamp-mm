0.3
Getting Started

0.3.1: Introduction

ဒီအပိုင်းမှာတော့ Boot Camp တစ်လျှောက် မင်းတို့ ဘာတွေသင်ယူရမယ်၊ ဘာတွေလုပ်ဆောင်ရမယ်ဆိုတာနဲ့ အထောက်အကူဖြစ်စေမယ့် Resources တွေကို လေ့လာသွားမှာပါ။

0.3.2: What You’ll Learn (ဘာတွေသင်ရမလဲ)

သင်တန်းကို အဓိက အဆင့် (၃) ဆင့် ခွဲထားပါတယ်။

Phase 1: Foundation (အခြေခံအဆင့်)

Front-end ပိုင်းနဲ့ ပတ်သက်တဲ့ အခြေခံတွေကို သင်ရမှာပါ။

  • HTML နှင့် CSS
  • Git (Code သိမ်းဆည်းနည်း)
  • JavaScript
  • Third-party APIs (jQuery, Bootstrap)
  • Server-side APIs, AJAX နှင့် JSON

ဒီအဆင့်မှာ ကိုယ်တိုင် Front-end Website တစ်ခု တည်ဆောက်ပြီး တခြားသူတွေရဲ့ APIs တွေကို ယူသုံးတတ်အောင် လေ့ကျင့်ရမှာပါ။

Phase 2: Technical (နည်းပညာအဆင့်)

Full-stack Web Application တစ်ခုလုံး တည်ဆောက်နိုင်အောင် သင်ရမှာပါ။ Back-end နည်းပညာတွေနဲ့ Database ပိုင်းတွေ ပါလာပါပြီ။

  • Node.js
  • Object-oriented programming (OOP)
  • Express.js (Servers)
  • MySQL (Database)
  • ORM (Sequelize)
  • MVC Framework (Handlebars.js)
  • Unit testing နှင့် Agile methodology

ဒီအဆင့်မှာ Front-end နဲ့ Back-end ကို ပေါင်းစပ်ပြီး Project တစ်ခု လုပ်ရပါမယ်။

Phase 3: Performance (စွမ်းဆောင်ရည်အဆင့်)

ပြုလုပ်ထားတဲ့ Web Apps တွေကို ပိုမြန်အောင်၊ ပိုကောင်းအောင် လုပ်နည်းတွေနဲ့ အလုပ်ခွင်ဝင်ဖို့ ပြင်ဆင်ရမယ့် အဆင့်ပါ။

  • Computer Science အခြေခံများ
  • MongoDB နှင့် NoSQL
  • Progressive Web Apps (PWAs)
  • React
  • MERN Stack
  • Portfolios ပြင်ဆင်ခြင်း

0.3.3: Boot Camp Expectations (လိုက်နာရမယ့် စည်းကမ်းများ)

  • တစ်ပတ်ကို စာကြည့်ချိန် ၂၀ နာရီကနေ ၃၀ နာရီ အထိ ပေးနိုင်အောင် ပြင်ဆင်ထားပါ။
  • Academic Honesty: Assignments နဲ့ Projects တွေက ကိုယ်တိုင်လုပ်ထားတဲ့ Original work ဖြစ်ရပါမယ်။
  • Assignments အားလုံးကို အချိန်မီတင်ပါ။ Challenge assignments ၂ ခုထက်ပို ပျက်ကွက်ရင် အမှတ်အပြည့် မရနိုင်ပါဘူး။
  • အချင်းချင်း ဆက်ဆံတဲ့အခါ ရိုရိုသေသေနဲ့ အားပေးတဲ့စကားလုံးတွေ သုံးပါ။

0.3.4: How to Succeed (အောင်မြင်ဖို့ နည်းလမ်းများ)

  • Overcome Frustration: ခက်ခဲတဲ့အခါ စိတ်မပျက်ပါနဲ့။ အချိန်ပေးပြီး လေ့ကျင့်ရင် တဖြည်းဖြည်း ကျွမ်းကျင်လာမှာပါ။
  • Take Charge: ကိုယ့်ဘာသာ တက်တက်ကြွကြွ လေ့လာပါ။ ဆရာတွေက လမ်းညွှန်ပေးရုံပဲမို့ ကိုယ်တိုင် အားထုတ်မှုက အရေးကြီးဆုံးပါ။
  • Art of Using Google: Coding မှာ အလွတ်ကျက်စရာမလိုပါဘူး။ Google မှာ အဖြေရှာတတ်ဖို့က အရေးကြီးတဲ့ Skill တစ်ခုပါ။
  • Self-Care: ကွန်ပျူတာရှေ့မှာ အကြာကြီးထိုင်ရမှာဖြစ်လို့ ခဏခဏ အကြောဆန့်ပေးပါ၊ နားနားနေနေ လုပ်ပါ။

0.3.5: Be Original (ကိုယ့်အားကိုယ်ကိုးပါ)

Plagiarism (သူများဆီက ခိုးယူကူးချခြင်း) ကို လုံးဝခွင့်မပြုပါဘူး။ အခက်အခဲရှိရင် Tutor ကို အကူအညီတောင်းပါ။

ခိုးချခြင်း (Plagiarism) ဆိုတာ ဘာလဲ?

  • သူများရေးထားတဲ့ စာသား ဒါမှမဟုတ် Code တွေကို ကိုယ့်ဟာလိုမျိုး တင်လိုက်တာ။
  • မူရင်းရေးသူကို Credit မပေးဘဲ သုံးတာ။

Code ကူးချတာမျိုး မဖြစ်အောင် ဘယ်လိုလုပ်မလဲ?

  • သူများဆီက Code အနည်းအကျဉ်း ယူသုံးရင် Comment ထဲမှာ Link ထည့်ပြီး Cite လုပ်ပေးပါ။
  • GitHub ကို ပုံမှန် Commit လုပ်ပါ။ ဒါမှ ကိုယ့်ရဲ့ Progress ကို သေချာပြနိုင်မှာပါ။
