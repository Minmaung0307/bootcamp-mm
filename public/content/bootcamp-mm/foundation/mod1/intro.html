Module 1: HTML, CSS, and Git ကို စတင်ခြင်း

၁၉၉၀ ခုနှစ်တွေတုန်းက ကမ္ဘာပေါ်မှာ ထူးခြားတဲ့ အပြောင်းအလဲတွေ အများကြီး ရှိခဲ့ပါတယ်။ အဲ့ဒီထဲမှာ အရေးအကြီးဆုံးကတော့ ဆယ်စုနှစ်ပေါင်းများစွာ တဖြည်းဖြည်း တိုးတက်လာခဲ့တဲ့ Internet ကနေ World Wide Web ဆိုတာ ပေါ်ပေါက်လာတာပါပဲ။

ဒီ Web ကြောင့်ပဲ သတင်းအချက်အလက်တွေကို အလွယ်တကူ မျှဝေနိုင်လာတယ်၊ ဖျော်ဖြေရေး ပုံစံသစ်တွေ ပေါ်လာတယ်၊ ကမ္ဘာ့စီးပွားရေး စနစ်သစ်တွေ ဖြစ်ပေါ်လာပြီး လူမှုကွန်ရက်ပေါ်က Cat memes တွေအထိ လူသားတွေရဲ့ နေထိုင်မှုပုံစံကို ပြောင်းလဲပေးခဲ့ပါတယ်။

ဒါတင်မကသေးပါဘူး၊ ဒီ Web ဟာ မင်းတို့အတွက် Web Developer တစ်ယောက် ဖြစ်လာဖို့ အနာဂတ် အလုပ်အကိုင် လမ်းကြောင်းကိုလည်း ဖန်တီးပေးလိုက်တာပါ။ ဒီ Module မှာဆိုရင် အခုလို ခေတ်သစ်ကို ဖြစ်ပေါ်စေခဲ့တဲ့ အခြေခံ နည်းပညာတွေဖြစ်တဲ့ HTML၊ CSS နဲ့ Git တို့ကို ကျွမ်းကျင်အောင် စတင် လေ့လာရမှာ ဖြစ်ပါတယ်။

ဒီအပတ်မှာ ဘာတွေ သင်ယူရမလဲဆိုတာ ပိုသိစေဖို့ အောက်က Video ကို ကြည့်လိုက်ရအောင်။
