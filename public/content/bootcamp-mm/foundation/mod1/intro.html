Introduction to Module 1

HTML, CSS, and Git

၁၉၉၀ ပြည့်လွန်နှစ်တွေမှာ ထူးခြားတဲ့အပြောင်းအလဲတွေ အများကြီး ရှိခဲ့ပါတယ်။ ဆယ်စုနှစ်ပေါင်းများစွာ တဖြည်းဖြည်း တိုးတက်လာခဲ့တဲ့ Internet ကနေ World Wide Web ဆိုတာ ပေါ်ပေါက်လာခဲ့တာပါ။ ဒီ Web ကြောင့်ပဲ သတင်းအချက်အလက်တွေကို အကြီးအကျယ် မျှဝေနိုင်လာတယ်၊ ဖျော်ဖြေရေး ပုံစံသစ်တွေ ပေါ်လာတယ်၊ ကမ္ဘာ့စီးပွားရေး စနစ်သစ်တွေ ဖြစ်ပေါ်လာသလို လူသားတွေရဲ့ နေထိုင်မှုပုံစံကို ပြောင်းလဲပေးခဲ့တဲ့ Cat memes တွေအထိ ပေါ်ပေါက်လာခဲ့ပါတယ်။

ဒီ Web နည်းပညာဟာ မင်းတို့အတွက် Web Developer တစ်ယောက် ဖြစ်လာဖို့ အနာဂတ် အလုပ်အကိုင် လမ်းကြောင်းကိုလည်း ဖန်တီးပေးလိုက်တာပါ။ ဒီ Module မှာဆိုရင် အခုလို ခေတ်သစ်ကို ဖြစ်ပေါ်စေခဲ့တဲ့ အခြေခံနည်းပညာတွေဖြစ်တဲ့ HTML၊ CSS နဲ့ Git တို့ကို ကျွမ်းကျင်အောင် စတင်လေ့လာရမှာ ဖြစ်ပါတယ်။

ဒီအပတ်မှာ ဘာတွေ မျှော်လင့်ထားလို့ရမလဲဆိုတာ ပိုသိစေဖို့ အောက်က Video ကို ကြည့်လိုက်ရအောင်။

ဒီအပတ်ရဲ့ Challenge

ဒီအပတ်အကုန်မှာ Challenge assignment တစ်ခုကို လုပ်ရပါမယ်။ အဲ့ဒါကတော့ ရှိပြီးသား webpage တစ်ခုကို Web Accessibility Standards (မသန်စွမ်းသူတွေပါ အဆင်ပြေပြေ သုံးနိုင်တဲ့ စံနှုန်းများ) နဲ့ ကိုက်ညီအောင် ပြင်ဆင်ရမှာပါ။ ဆိုလိုတာက အမြင်အာရုံ ချို့တဲ့သူတွေသုံးတဲ့ screen readers လိုမျိုး assistive technology တွေနဲ့ ဖတ်လို့ရအောင် လုပ်ပေးရမှာ ဖြစ်သလို၊ code တွေကိုလည်း ပိုပြီး သေသပ်ကောင်းမွန်အောင် ရှင်းလင်းရမှာ ဖြစ်ပါတယ်။

ဒီ challenge ကို အောင်အောင်မြင်မြင် လုပ်နိုင်ဖို့ အောက်ပါ skills တွေကို အသုံးချရပါမယ် -

  • HTML နဲ့ CSS ရဲ့ ကွာခြားချက်၊ တစ်ခုချင်းစီရဲ့ အလုပ်လုပ်ပုံနဲ့ ၎င်းတို့ အပြန်အလှန် ချိတ်ဆက်ပုံကို သိရမယ်။
  • Semantic HTML အကြောင်းနဲ့ HTML files တွေကို စနစ်တကျ တည်ဆောက်တဲ့ best practices တွေကို နားလည်ရမယ်။
  • Screen readers တွေက page ရဲ့ အကြောင်းအရာကို သေချာဖတ်ပြနိုင်အောင် accessibility standards နဲ့အညီ HTML elements တွေကို ဖွဲ့စည်းရမယ်။
  • CSS files တွေကို စနစ်တကျ ရေးသားတဲ့ နည်းလမ်းတွေကို လိုက်နာရမယ်။
  • Code ကို ဖတ်ရလွယ်အောင် CSS files ထဲမှာ comments တွေ ထည့်တတ်ရမယ်။
  • ကိုယ့်ရဲ့ webpage ကို အင်တာနက်ပေါ် တင်ဖို့ (publish လုပ်ဖို့) GitHub Pages ကို သုံးတတ်ရမယ်။

ဒါတွေက အခုလောလောဆယ် အစိမ်းသက်သက် ဖြစ်နေရင်လည်း မစိုးရိမ်ပါနဲ့။ ဒီအပတ်ရဲ့ သင်ခန်းစာတွေမှာ Run Buddy ဆိုတဲ့ landing page လေးကို တည်ဆောက်ရင်း ဒါတွေကို တစ်ဆင့်ချင်း သင်ယူသွားရမှာပါ။

မင်းတို့ ဘာတွေ သင်ယူရမလဲ?

ဒီ module အပြီးမှာတော့ အောက်ပါအချက်တွေကို တတ်မြောက်သွားမှာပါ -

  • Semantic HTML နဲ့ CSS ကို သုံးပြီး static webpages တွေ တည်ဆောက်တတ်မယ်။
  • HTML files တွေ ရေးတဲ့အခါ nested elements၊ indentation၊ comments နဲ့ line breaks စတဲ့ စံနှုန်းတွေကို လိုက်နာတတ်မယ်။
  • External stylesheets၊ inline styles နဲ့ embedded style tags တွေကို သုံးပြီး class၊ id နဲ့ element selectors တွေနဲ့ CSS styling လုပ်တတ်မယ်။
  • Cascading Style Sheets ရဲ့ “cascade” အလုပ်လုပ်ပုံကို ရှင်းပြနိုင်မယ်။
  • CSS Box Model ကို သုံးပြီး space နဲ့ margin တွေကို သတ်မှတ်တတ်မယ်။
  • Version control ရဲ့ အရေးပါပုံကို သိပြီး Git workflow သုံးကာ projects တွေ စတင်တာ၊ changes တွေကို မှတ်တမ်းတင်တာနဲ့ remote server ပေါ် တင်တာတွေ လုပ်တတ်မယ်။
  • Command line သုံးပြီး operating system နဲ့ ချိတ်ဆက် ခိုင်းစေတတ်မယ်။
  • GitHub Pages သုံးပြီး live deployment (အင်တာနက်ပေါ် တင်ခြင်း) လုပ်တတ်မယ်။
  • HTML နဲ့ CSS issues တွေကို ဘယ်လို debug လုပ်ရမလဲဆိုတာ သိမယ်။
  • HTML attributes တွေအကြောင်း၊ အထူးသဖြင့် form elements တွေမှာ ဘယ်လို သက်ရောက်မှု ရှိလဲဆိုတာ ရှင်းပြနိုင်မယ်။
  • Positioning နဲ့ floats တွေကို သုံးပြီး CSS layouts တွေကို ထိန်းချုပ်တတ်မယ်။

Weekly Tip: သင်တန်းကို ကိုယ်ပိုင် တန်ဖိုးထားမှုတွေနဲ့ ချိတ်ဆက်ပါ

ပထမဆုံး အပတ်မှာတော့ Personal Values (ကိုယ်ပိုင် တန်ဖိုးထားမှုများ) အကြောင်း ပြောချင်ပါတယ်။ ကျွန်တော်တို့မှာ ဘဝရဲ့ ဆုံးဖြတ်ချက်တွေကို လွှမ်းမိုးနေတဲ့ မိသားစု၊ အောင်မြင်မှု၊ တီထွင်ဖန်တီးနိုင်မှု စတဲ့ တန်ဖိုးထားမှုတွေ ရှိကြပါတယ်။

ဒီသင်တန်းကို ဘာကြောင့် တက်ဖို့ ဆုံးဖြတ်ခဲ့တာလဲဆိုတာကို ကိုယ့်ရဲ့ တန်ဖိုးထားမှုတွေနဲ့ ချိတ်ဆက်ကြည့်ရင် ပင်ပန်းတဲ့အခါမှာလည်း နောက်မဆုတ်ဘဲ ဆက်သွားနိုင်ဖို့ အားဖြစ်စေပါတယ်။ ဥပမာ -

  • Adventurousness (စွန့်စားလိုစိတ်) ကို တန်ဖိုးထားရင် နည်းပညာအသစ်တွေကို လေ့လာရတာဟာ ခရီးသစ်တစ်ခု သွားနေရသလို ခံစားရပါလိမ့်မယ်။
  • Family (မိသားစု) ကို တန်ဖိုးထားရင် ဒီပညာရပ်က ကိုယ့်မိသားစုကို ပိုကောင်းတဲ့ ဘဝတစ်ခု ပေးနိုင်မယ်လို့ တွေးပြီး ကြိုးစားနိုင်ပါတယ်။

အချိန်ဇယား ဆွဲခြင်း

ဒီ module မှာ သင်ခန်းစာ ၇ ခု ပါဝင်ပါတယ်။ အချိန်ကို အောက်ပါအတိုင်း စီစဉ်ထားပါ -

  • ပထမဆုံး Virtual Class မတိုင်ခင် သင်ခန်းစာ (Lesson) ၁ နဲ့ ၂ ကို အပြီးလေ့လာထားပါ။
  • ဒုတိယမြောက် Virtual Class မတိုင်ခင် သင်ခန်းစာ ၃၊ ၄ နဲ့ ၅ ကို အပြီးလေ့လာထားပါ။
  • Challenge assignment ကို အပတ်စဉ် အစကတည်းက ကြိုကြည့်ထားပါ။ ကြိုက်တဲ့အချိန်မှာ စလုပ်လို့ ရပါတယ်။

ဒီပထမဆုံး module ကတော့ အသစ်အဆန်းတွေ၊ tools တွေ အများကြီး မိတ်ဆက်ရမှာ ဖြစ်လို့ နည်းနည်း ပိုရှည်ပါလိမ့်မယ်။ နောက်ပိုင်း modules တွေမှာတော့ သင်တန်းသားတွေ လက်ယဉ်သွားမှာ ဖြစ်လို့ ပိုပြီး တိုတိုရှင်းရှင်း ဖြစ်လာမှာပါ။

Getting Ready (စတင်ဖို့ ပြင်ဆင်ကြစို့)

ဒီ Module မှာ အသုံးပြုမယ့် tools တွေရဲ့ စာရင်းပါ။ အကယ်၍ prework လုပ်ခဲ့ရင် အချို့က install လုပ်ပြီးသား ဖြစ်နေပါလိမ့်မယ်။ အသစ်တွေဆိုရင်တော့ သင်ခန်းစာထဲမှာမှ တစ်ဆင့်ချင်း install လုပ်သွားမှာမို့ အခုချက်ချင်း လုပ်စရာမလိုသေးပါဘူး။

အသုံးပြုမယ့် tools များမှာ -

  • Google Chromeproject ကို စမ်းသပ်ဖို့နဲ့ debug လုပ်ဖို့ အဓိက သုံးမယ့် web browser ဖြစ်ပါတယ်။
  • Chrome DevTools: Google Chrome ထဲမှာတင် ပါပြီးသား tools တွေပါ။ web page တွေကို ချက်ချင်းပြင်ကြည့်တာနဲ့ ပြဿနာတွေကို ရှာဖွေ (debug) တာတွေ လုပ်နိုင်ပါတယ်။
  • The command line: graphical interface (ပုံတွေ၊ ခလုတ်တွေ) မလိုဘဲ စာသားနဲ့ပဲ project တွေ တည်ဆောက်ဖို့နဲ့ ကွန်ပျူတာကို ခိုင်းစေဖို့ သုံးပါတယ်။
  • Git: ကိုယ်လုပ်ထားတဲ့ အလုပ်တွေကို မှတ်တိုင် (save points) အနေနဲ့ သိမ်းထားဖို့ သုံးတဲ့ စနစ်ပါ။ ဒါကို commits လို့ ခေါ်ပါတယ်။ လိုအပ်ရင် အရင် version ဆီ ပြန်သွားလို့လည်း ရပါတယ်။
  • Visual Studio (VS) Code:code တွေ ရေးဖို့ အခမဲ့ သုံးနိုင်တဲ့ text editor ဖြစ်ပါတယ်။
  • GitHub:Git ကို Windows မှာ သုံးဖို့ Git repositories တွေကို အင်တာနက်ပေါ်မှာ သိမ်းပေးတဲ့ website ဖြစ်ပါတယ်။ အခြားသူတွေနဲ့ code တွေ စုပေါင်းရေးသားဖို့လည်း သုံးပါတယ်။
  • GitHub Pages: GitHub ပေါ်က HTML, CSS နဲ့ JavaScript ဖိုင်တွေကို live website တစ်ခုအဖြစ် အင်တာနက်ပေါ် တင်ပေးတဲ့ static site hosting service ဖြစ်ပါတယ်။
