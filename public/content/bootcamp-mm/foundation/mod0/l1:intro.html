Introduction to the Coding Boot Camp

Blended Online Coding Boot Camp ကို ကြိုဆိုပါတယ်! ဒီ program က လွယ်မလွယ်ပေမယ့် တကယ်အကျိုးရှိပြီး full-stack web developer ဖြစ်ဖို့ လိုအပ်တဲ့ technical skills နဲ့ mindset ကို တည်ဆောက်ပေးမယ့် course တစ်ခုပါ။

အပတ်တိုင်း coding languages, web technologies, software development concepts တွေကို တဖြည်းဖြည်း လေ့လာသွားမယ်။ Foundation → Technical → Performance ဆိုပြီး phase ၃ ခုနဲ့ သင်ယူပြီး phase တစ်ခုပြီးတိုင်း real-world problem ကို ဖြေရှင်းနိုင်တဲ့ project တစ်ခုစီ ဆောက်ရမယ်။

Community support လည်းရှိတော့ မသိတာတွေ၊ အခက်အခဲတွေကို အတူကူညီနိုင်ပါတယ်။ ကြိုးစားအားထုတ်ရင် boot camp student ကနေ professional full-stack web developer ဖြစ်လာနိုင်တယ် 👍

ဒီ introductory module က Coding Boot Camp ကို စတင်မယ့်သူတွေအတွက် အရင်ဆုံး သိထားရမယ့် အချက်အလက်တွေကို မိတ်ဆက်ပေးတဲ့ module ပါ။ Online learning ကို ဘယ်လိုသုံးရမလဲ၊ tools တွေကို ဘယ်လို အသုံးချရမလဲဆိုတာတွေကို activity လေးတွေကနေ တဖြည်းဖြည်း သိလာအောင် လုပ်ထားတာပါ။

ဒီ module ထဲမှာ course objectives (သင်တန်းရဲ့ ရည်ရွယ်ချက်တွေ) expectations (ဘာတွေ မျှော်လင့်ထားလဲ) boot camp ကို အောင်မြင်အောင် ဘယ်လို လေ့လာသင့်လဲဆိုတဲ့ strategies တွေကို သင်ယူရမယ်။

ပြီးရင် လိုအပ်တဲ့ tools တွေကို install လုပ်ရမယ်၊ ကိုယ့်ရဲ့ hardware နဲ့ software တွေက technical requirements တွေနဲ့ ကိုက်ညီမညီ စစ်ဆေးရမယ်။ ဒီ module ကို ပြီးသွားရင် Day 1 ကို စတင်ဖို့ အဆင်သင့် ဖြစ်သွားပြီး အောက်ပါအကြောင်းအရာတွေကို သိထားရမယ် –

  • Coding Boot Camp ကို အောင်မြင်အောင် သင်တန်းကာလအတွင်း schedule ကို ဘယ်လို စီမံရမလဲ
  • Coding Boot Camp ရဲ့ course structure က ဘယ်လို ဖွဲ့စည်းထားလဲ
  • Support team ရဲ့ role က ဘာလဲ၊ support resources တွေကို boot camp တလျှောက် ဘယ်လို ရယူနိုင်လဲ
  • Zoom နဲ့ live class session တွေ တက်တဲ့အခါ ဘာတွေ မျှော်လင့်ရမလဲ
  • Bootcamp Spot learning platform ကို tour နဲ့ orientation လုပ်ပေးမယ်
  • Zoom orientation – Office Hours, Tutor Support, Live Classroom တွေအတွက် Zoom ကို ဘယ်လို သုံးမလဲ
  • Hardware နဲ့ software requirements တွေ
  • Module 1 စတင်ဖို့ install လုပ်ထားရမယ့် tools တွေ

Module Objectives

ဒီ module ကို ပြီးသွားရင် အောက်ပါအရာတွေကို လုပ်နိုင်လာမယ် –

  • Bootcamp Spot interface နဲ့ collaborative learning technologies တွေကို သေချာ အသုံးပြုနိုင်မယ်
  • ကိုယ့်ရဲ့ technology (computer, software, internet) တွေက minimum requirements နဲ့ ကိုက်ညီမညီ စစ်ဆေးနိုင်ပြီး Module 1 မှာ project စလုပ်ဖို့ လိုအပ်တဲ့ tools တွေကို install လုပ်ထားနိုင်မယ်
  • Coding Boot Camp ရဲ့ phase ၃ ခုကို နာမည်တပ်ပြောနိုင်ပြီး schedule နဲ့ requirements တွေကို ရှင်းပြနိုင်မယ်
  • အောင်မြင်ဖို့ strategies တွေကို သိနိုင်မယ်၊ assignments နဲ့ projects တွေကို ပြီးအောင်လုပ်နိုင်မယ်၊ လိုအပ်ရင် support ကို ဘယ်လို ရယူရမလဲ သိလာမယ်

Navigating This Course

သင်တန်းစဖို့ စိတ်လှုပ်ရှားနေပြီဆိုတာ သိပါတယ် 😄 မစခင် ဒီ course ကို ဘယ်လို သုံးရမလဲ၊ အပတ်တိုင်း ဘာတွေ မျှော်လင့်ရမလဲဆိုတာ အရင်ဆုံး နည်းနည်းလေး ရှင်းပြချင်ပါတယ်။

What to Expect in Each Module

ဒီ course ရဲ့ module တစ်ခုစီကို topic တစ်ခုနဲ့ skill set တစ်ခုစီအပေါ် အခြေခံပြီး စီထားပါတယ်။ Module တစ်ခုထဲမှာ အောက်ပါအရာတွေ ပါပါတယ် – • Online lessons အချို့ • Instructor-led classes ၂ ခါ (Zoom နဲ့) • Challenge assignment သို့မဟုတ် group project တစ်ခု

  • Online lessons တွေကို ကိုယ်အချိန်ရတဲ့အခါ ကိုယ်တိုင် လေ့လာရတာပါ။ Class မတက်ခင် အဲ့ဒီ class နဲ့ ဆိုင်တဲ့ lessons တွေကို အရင် ပြီးအောင် လုပ်ထားရင် class ထဲမှာ activity တွေကို ပိုပြီး ပါဝင်နိုင်မယ်၊ အကျိုးလည်း ပိုရမယ်။
  • Module တစ်ခုစီရဲ့ Introduction ထဲမှာရှိတဲ့ Roadmap page ကို ကြည့်ရင် ဘယ် class မတက်ခင် ဘယ် lessons တွေ ပြီးထားရမလဲဆိုတာ အသေးစိတ် ဖော်ပြထားပါတယ်။
  • First module အတွက်တော့ ပထမ virtual class နဲ့ ဆိုင်တဲ့ lessons တွေ ပြီးမပြီး စစ်နိုင်အောင် quick survey လေးပါ ထည့်ထားပါတယ်။

အပတ်တိုင်း Zoom နဲ့ instructor-led virtual class ၂ ခါ ရှိပါတယ်။ ဒီ classes တွေက online lessons ပေါ်မှာ အခြေခံပြီး ဆက်သွားတာမို့ lessons ကို တော်တော်လေး လုပ်ထားမှ class ကို အပြည့်အဝ အသုံးချနိုင်မယ်။

Course navigation menu ထဲက Zoom link ကနေ virtual class meetings ကို ဝင်လို့ရပါတယ်။

Challenge Assignment / Group Project

အပတ်ရဲ့ နောက်ဆုံးမှာ သင်လေ့လာခဲ့တဲ့ skills တွေကို ပြသဖို့ Challenge assignment သို့မဟုတ် group project တစ်ခု submit လုပ်ရမယ်။ ဒီ Challenges နဲ့ projects တွေက graded assignments ဖြစ်ပြီး feedback ကိုလည်း ရမှာပါ။

Assignment နဲ့ virtual class date တွေ မမေ့အောင် class calendar ကို ဖုန်း calendar သို့မဟုတ် web calendar နဲ့ sync လုပ်ထားတာ အရမ်းအသုံးဝင်ပါတယ်။ Sync လုပ်နည်းကတော့ –

  • Main navigation bar ထဲက calendar icon ကို နှိပ်ပြီး course calendar ကို ဝင်ပါ
  • Calendar Feed link ကို ရှာပြီး iCal link ကို ယူပါ
  • အဲ့ဒီ link ကို Google Calendar, iCal, Outlook စတဲ့ iCal support ပေးတဲ့ calendar app ထဲကို paste လုပ်ပါ
