Introduction to the Coding Boot Camp

0.1.1: Welcome to the Coding Boot Camp

Blended Online Coding Boot Camp ကနေ ကြိုဆိုပါတယ်! ဒီသင်တန်းဟာ ခက်ခဲပေမဲ့ တန်ဖိုးရှိတဲ့ သင်တန်းတစ်ခုဖြစ်ပြီး မင်းကို Full-stack Web Developer တစ်ယောက်ဖြစ်လာဖို့ လိုအပ်တဲ့ Technical Skills တွေနဲ့ Mindset တွေရအောင် အသေအချာ လေ့ကျင့်ပေးသွားမှာပါ။ Real-life Scenarios (လက်တွေ့နယ်ပယ်က အခြေအနေများ) ကို အခြေခံပြီး သင်ယူရမှာဖြစ်သလို အချင်းချင်း ကူညီပေးမယ့် Community တစ်ခုလည်း ရှိနေမှာပါ။

အပတ်တိုင်းမှာ Coding Languages၊ Web Technologies နဲ့ Software Development အယူအဆတွေကို ကျွမ်းကျင်အောင် သင်ယူရပါမယ်။ သင်တန်းမှာ အဓိက အဆင့် (၃) ဆင့် ရှိပါတယ် -

  • Foundation (အခြေခံအဆင့်)
  • Technical (နည်းပညာအဆင့်)
  • Performance (စွမ်းဆောင်ရည်အဆင့်)

အဆင့်တစ်ခုစီ အပြီးတိုင်းမှာ လက်တွေ့အသုံးဝင်မယ့် Project တစ်ခုစီကို ကိုယ်တိုင် ဖန်တီးရမှာပါ။ မင်းဘက်က ကြိုးစားအားထုတ်မှုအပြည့်ရှိမယ်ဆိုရင် ကျောင်းသားဘဝကနေ အောင်မြင်တဲ့ Professional Full-stack Web Developer တစ်ယောက်အဖြစ် ပြောင်းလဲသွားမှာ သေချာပါတယ်။

0.1.2: Introduction

ဒီ Introductory Module ကတော့ သင်တန်းအကြောင်း အကျဉ်းချုပ်နဲ့ အွန်လိုင်းမှာ သင်ယူရမယ့် ပုံစံ၊ အသုံးပြုမယ့် Tools တွေကို ရင်းနှီးသွားအောင် မိတ်ဆက်ပေးမှာပါ။ သင်တန်းရဲ့ ရည်ရွယ်ချက်၊ လိုက်နာရမယ့် စည်းကမ်းတွေနဲ့ အောင်မြင်အောင် ဘယ်လိုလုပ်ရမလဲဆိုတာတွေကို သိရမှာ ဖြစ်သလို၊ လိုအပ်တဲ့ Tools တွေကို Install လုပ်ပြီး Technical Requirements (နည်းပညာပိုင်းဆိုင်ရာ လိုအပ်ချက်များ) ပြည့်မီခြင်း ရှိမရှိကို စစ်ဆေးသွားမှာပါ။

ဒီ Module ကို လေ့လာပြီးသွားရင် မင်းတို့အနေနဲ့ Day 1 ကို စတင်ဖို့ အသင့်ဖြစ်သွားပါပြီ။ အောက်ပါ အကြောင်းအရာတွေကို သိရှိသွားမှာ ဖြစ်ပါတယ် -

  • Coding Boot Camp မှာ အောင်မြင်ဖို့ Schedule (အချိန်ဇယား) ကို ဘယ်လို စနစ်တကျ ဆွဲမလဲ။
  • သင်တန်းရဲ့ Course Structure (ဖွဲ့စည်းပုံ) အကြောင်း။
  • Support Team ရဲ့ အခန်းကဏ္ဍနဲ့ အကူအညီတွေကို ဘယ်လို ရယူမလဲ။
  • Zoom နဲ့ Live Class တက်တဲ့အခါ ဘာတွေ သိထားရမလဲ။
  • Bootcamp Spot Learning Platform ကို လှည့်လည်ကြည့်ရှုခြင်းနဲ့ မိတ်ဆက်။
  • Office Hours၊ Tutor Support နဲ့ Live Classroom တွေအတွက် Zoom အသုံးပြုပုံ။
  • Hardware နဲ့ Software လိုအပ်ချက်များ။
  • Module 1 စတင်ဖို့အတွက် Install လုပ်ထားရမယ့် Tools များ။

Module Objectives

ဒီ Module အပြီးမှာတော့ မင်းတို့အနေနဲ့ အောက်ပါအချက်တွေကို လုပ်ဆောင်နိုင်မှာ ဖြစ်ပါတယ် -

  • Bootcamp Spot Interface နဲ့ သင်ယူမှုမှာ အသုံးပြုမယ့် နည်းပညာတွေကို ကျွမ်းကျင်စွာ အသုံးပြုတတ်သွားမယ်။
  • ကိုယ့်ရဲ့ Technology (စက်ပစ္စည်း) တွေက Minimum Requirements (အနိမ့်ဆုံး လိုအပ်ချက်) ပြည့်မီကြောင်း စစ်ဆေးနိုင်မယ်။ ပြီးတော့ Module 1 မှာ Project စတင်ဖို့ လိုအပ်တဲ့ Tools တွေကို Install လုပ်တတ်သွားမယ်။
  • သင်တန်းမှာ ပါဝင်တဲ့ Phase ၃ ခုအကြောင်းနဲ့ အချိန်ဇယား၊ လိုအပ်ချက်တွေကို ရှင်းပြနိုင်မယ်။
  • သင်တန်းမှာ အောင်မြင်ဖို့ နည်းဗျူဟာတွေ၊ Assignments နဲ့ Projects တွေ တင်နည်းတွေအပြင် အကူအညီ (Support) ဘယ်လို တောင်းရမလဲဆိုတာ သိသွားမယ်။

0.1.3: Navigating This Course (သင်ခန်းစာတွေကို ကြည့်ရှုပုံ)

သင်တန်းမစခင်မှာ အပတ်စဉ် ဘာတွေလုပ်ရမလဲဆိုတာနဲ့ သင်ခန်းစာတွေကို ဘယ်လို လေ့လာရမလဲဆိုတဲ့ အချက်အလက်တချို့ကို သိထားဖို့ လိုပါတယ်။

Module တစ်ခုစီမှာ ဘာတွေပါမလဲ?

Module တစ်ခုစီမှာ သတ်မှတ်ထားတဲ့ ခေါင်းစဉ်တစ်ခုကို ကျွမ်းကျင်အောင် သင်ယူရမှာပါ။ Module တိုင်းမှာ အောက်ပါအတိုင်း ဖွဲ့စည်းထားပါတယ် -

  • Online Lessons: ကိုယ်ပိုင်အချိန်မှာ လေ့လာရမယ့် သင်ခန်းစာတွေပါ။ Virtual Class မစခင် ဒါတွေကို အပြီးကြည့်ထားမှ အတန်းထဲမှာ အဆင်ပြေပြေ လိုက်ပါနိုင်မှာပါ။
  • Virtual Classes: တစ်ပတ်ကို ၂ ကြိမ် Zoom ကနေ ဆရာနဲ့ တိုက်ရိုက်သင်ရမှာပါ။ Online Lessons တွေကို အခြေခံပြီး သင်မှာဖြစ်လို့ ကြိုတင်လေ့လာထားဖို့ အရေးကြီးပါတယ်။
  • Challenge Assignment / Group Project: တစ်ပတ်စာ သင်ထားတာတွေကို လက်တွေ့လုပ်ပြရမယ့် အပိုင်းပါ။ ဒါကို Grade (အမှတ်) ပေးမှာဖြစ်သလို ဆရာတွေဆီက Feedback လည်း ပြန်ရပါမယ်။

Class Calendar ကို Sync လုပ်နည်း:

အချိန်ဇယားတွေကို ဖုန်းထဲမှာ အလွယ်တကူ ကြည့်နိုင်ဖို့အတွက် အောက်ပါအတိုင်း လုပ်ဆောင်ပါ -

  • Google Calendar၊ iCal ဒါမှမဟုတ် Outlook ထဲမှာ မှတ်သားထားပါ။

Online သင်ခန်းစာတွေကို ဘယ်လိုကြည့်မလဲ?

Module တစ်ခုဖွင့်တိုင်း ပထမဆုံး အောက်ပါ စာမျက်နှာ ၄ ခုကို အရင်ကြည့်ရပါမယ် -

  • Introduction to Module: ဒီအပတ်မှာ ဘာတွေသင်ရမလဲဆိုတဲ့ အနှစ်ချုပ်။
  • Roadmap: အရင်သင်ခဲ့တာတွေနဲ့ အခုသင်မှာတွေ ဘယ်လိုဆက်စပ်လဲဆိုတာကို ပြထားပါတယ်။ Virtual Class မတိုင်ခင် ဘယ်သင်ခန်းစာတွေ ပြီးအောင်ကြည့်ရမလဲဆိုတာ ဒီမှာ ကြည့်နိုင်ပါတယ်။
  • Getting Ready for Class: အတန်းထဲမှာသုံးမယ့် Starter Files တွေကို Download ဆွဲထားရမယ့်နေရာပါ။
  • Up and Running: အပတ်စဉ်လုပ်ရမယ့် Module Project အကြောင်းနဲ့ လိုအပ်တဲ့ Tools/Technologies အသစ်တွေကို Install လုပ်နည်းတွေ ပါပါတယ်။

ဒီမိတ်ဆက်စာမျက်နှာတွေပြီးရင်တော့ Online Lessons တွေကို တစ်မျက်နှာချင်းစီ အစဉ်လိုက် လေ့လာသွားရုံပါပဲ။ Module တစ်ခုရဲ့ နောက်ဆုံးမှာတော့ Challenge တင်ရမယ့်နေရာနဲ့ လိုအပ်တဲ့ Resources တွေကို တွေ့ရပါလိမ့်မယ်။
