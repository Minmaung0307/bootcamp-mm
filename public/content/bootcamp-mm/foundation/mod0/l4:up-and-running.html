0.4.1: Introduction

ဒီသင်တန်းမှာ ပါဝင်ဖို့အတွက် မင်းတို့ရဲ့ assignments တွေနဲ့ projects တွေ တင်ရမယ့် Bootcamp Spot ဆိုတဲ့ learning platform အကြောင်း အခြေခံလေးတွေ သိထားဖို့ လိုပါတယ်။ အသေးစိတ် အချက်အလက်တွေ သိချင်ရင်လည်း ထပ်မံလေ့လာနိုင်တဲ့ resources တွေကို ပေးထားပါတယ်။

အခုလောလောဆယ်တော့ အောက်ပါအချက်တွေကို သိထားရင် ရပါပြီ -

  • သင်ခန်းစာတွေကို လေ့လာဖို့ Bootcamp Spot ထဲမှာ ဘယ်လို သွားလာအသုံးပြုရမလဲ (Navigate လုပ်နည်း)။
  • ကိုယ့်ရဲ့ Challenges တွေနဲ့ projects တွေကို ဘယ်လို တင်ရမလဲ (Submit လုပ်နည်း)။
  • Bootcamp Spot မှာ ကိုယ့်ရဲ့ profile ကို ပြင်ဆင်တာနဲ့ ဓာတ်ပုံထည့်တာတွေကို ဘယ်လိုလုပ်ရမလဲ။

0.4.2: Bootcamp Spot Overview

သင်တန်းမှာ အသုံးပြုမယ့် Bootcamp Spot (Learning Platform) အကြောင်း အခြေခံလေးတွေ သိထားဖို့ လိုပါတယ်။ သင်တန်းတက်ရင်းနဲ့ ဒီစနစ်ကို ပိုပြီး ကျွမ်းကျင်လာမှာဖြစ်သလို အသေးစိတ် သိချင်ရင်လည်း Official documentation မှာ ဖတ်နိုင်ပါတယ်။

Submit Assignments (အလုပ်များ တင်ခြင်း)

Challenges နဲ့ Projects တွေကို Bootcamp Spot ပေါ်မှာ တိုက်ရိုက် တင်ရပါမယ် -

  1. Assignment စာမျက်နှာကို သွားပြီး ညွှန်ကြားချက်တွေကို အရင်ဖတ်ပါ။
  2. တင်ဖို့ အဆင်သင့်ဖြစ်ရင် Submit Assignment ခလုတ်ကို နှိပ်ပါ။
  3. တင်ရမယ့် ပုံစံ ၃ မျိုး ရှိနိုင်ပါတယ် -
    • Text Entry: စာရိုက်ပြီး တင်ရတဲ့ ပုံစံ။
    • Website URL: GitHub link လိုမျိုး Link ပေးပြီး တင်ရတဲ့ ပုံစံ။
    • Choose File: ကိုယ့်စက်ထဲက ဖိုင်ကို Upload တင်ရတဲ့ ပုံစံ။
  4. ဖြည့်စွက်ပြီးရင် Submit Assignment ခလုတ်ကို နှိပ်ပြီး အဆုံးသတ်ပါ။

0.4.4: Slack and Zoom

သင်တန်းတစ်လျှောက်မှာ အတန်းဖော်တွေ၊ ဆရာတွေနဲ့ ဆက်သွယ်ဖို့ Slack နဲ့ Zoom ကို အဓိက သုံးသွားမှာပါ။

Slack

Slack ဆိုတာ သင်တန်းသားအချင်းချင်း စာသင်ချိန်ပြင်ပမှာပါ စကားပြောလို့ရတဲ့ Virtual Workspace ဖြစ်ပါတယ်။ ဒီမှာ ဗဟုသုတတွေ ဝေမျှတာ၊ Study Group ဖွဲ့တာတွေ လုပ်နိုင်ပါတယ်။ Slack ကို ကွန်ပျူတာမှာရော ဖုန်းမှာပါ Install လုပ်ပြီး သုံးလို့ရပါတယ်။

Slack ကို အကျိုးရှိရှိ သုံးနိုင်ဖို့ နည်းလမ်းများ -

  • Notifications ဖွင့်ထားပါ: သတင်းအချက်အလက်တွေ မလွတ်သွားအောင် Channel တစ်ခုချင်းစီရဲ့ Settings မှာ အသိပေးချက်တွေ ဖွင့်ထားပါ။
  • App ကို Download ဆွဲသုံးပါ: Web browser ကနေ သုံးမယ့်အစား Desktop App နဲ့ Mobile App ကို သုံးတာက ပိုပြီး အဆင်ပြေပါတယ်။
  • AskBCS bot ကို သုံးပါ: အတန်းချိန်ပြင်ပမှာ နည်းပညာပိုင်းဆိုင်ရာ အကူအညီလိုရင် AskBCS bot ကို သုံးပြီး မေးခွန်းမေးပါ။
  • Threads နဲ့ ပြန်ဖြေပါ: စာတွေ ရှုပ်ပွမသွားအောင် မက်ဆေ့ချ်တစ်ခုကို ပြန်ဖြေတဲ့အခါ Reply in thread ကို သုံးပါ။
  • Channels အသစ်ဆောက်ပါ: စာစုပြီး လေ့လာဖို့အတွက် ကိုယ်ပိုင် Channels တွေ အသစ်ဆောက်ပြီး အတန်းဖော်တွေကို ဖိတ်ခေါ်နိုင်ပါတယ်။
  • သတိပြုရန်: TAs တွေကို Direct Message (DM) ပို့ပြီး နည်းပညာအကူအညီ မတောင်းပါနဲ့။ AskBCS သို့မဟုတ် Office Hours ကိုပဲ အသုံးပြုပါ။

Zoom

အပတ်စဉ် Live Class တွေနဲ့ Office Hours တွေကို Zoom နဲ့ လုပ်သွားမှာပါ။ ဗီဒီယိုကနေတစ်ဆင့် ဆရာတွေနဲ့ တိုက်ရိုက်တွေ့ဆုံနိုင်သလို Chatbox ကနေလည်း စာရိုက်ပြီး မေးနိုင်ပါတယ်။

Zoom သုံးတဲ့အခါ လိုက်နာရန် အချက်များ -

  • မိမိစကားမပြောတဲ့အချိန်မှာ Microphone ကို Mute (အသံပိတ်) ထားပါ။
  • အချင်းချင်း ဆက်သွယ်မှု ပိုကောင်းအောင် သင်တန်းချိန်တစ်လျှောက်လုံး Camera ဖွင့်ထားပေးပါ။
  • Zoom Name နေရာမှာ မိမိရဲ့ နာမည်အပြည့်အစုံကို သုံးပါ။
  • ဆရာတွေက Polls (မဲခွဲမေးမြန်းခြင်း) တွေ လုပ်တဲ့အခါ တက်တက်ကြွကြွ ပါဝင်ပေးပါ။

0.4.5: Equipment Requirements (လိုအပ်တဲ့ စက်ပစ္စည်းများ)

ဒီသင်တန်းမှာပါတဲ့ tools တွေနဲ့ နည်းပညာတွေကို ကောင်းကောင်းအသုံးပြုနိုင်ဖို့ အောက်ပါစက်ပစ္စည်းတွေ လိုအပ်ပါတယ် -

  • Laptop: အနည်းဆုံး 8 GB RAM ရှိပြီး 64-bit dual-core processor အမျိုးအစား ဖြစ်ရပါမယ်။
  • High-speed internet: အင်တာနက်လိုင်း ကောင်းဖို့လိုပါတယ်။ (အနည်းဆုံး download speed 25 Mbps နဲ့ upload speed 5 Mbps ရှိဖို့ အကြံပြုပါတယ်)။

ပထမဆုံး virtual class မစခင် အောက်ပါပစ္စည်းတွေကို အဆင်သင့်ပြင်ထားပါ -

  • Webcam (ကင်မရာ)
  • Headphones with a microphone (နားကြပ်နှင့် မိုက်ခရိုဖုန်း)

နောက်ထပ် external monitor (နောက်ထပ် စခရင်တစ်ခု) ရှိရင်တော့ coding လုပ်ရတာ ပိုပြီး အဆင်ပြေလွယ်ကူစေမှာဖြစ်လို့ အသုံးပြုဖို့ အထူးတိုက်တွန်းလိုပါတယ်။

0.4.6: Install Your Tools

သင်တန်းတစ်လျှောက်မှာ လိုအပ်တဲ့ tools တွေကို install လုပ်ခိုင်းတာ ဒါမှမဟုတ် services တွေမှာ sign up လုပ်ခိုင်းတာမျိုးတွေ ရှိပါမယ်။ အကယ်၍ prework တွေ မလုပ်ရသေးရင် Module 01 မစခင် အခုပြောမယ့်အရာတွေကို အပြီး install လုပ်ထားဖို့ လိုပါတယ်။

Ready for Action

Day 1 အတန်းစတဲ့အခါ အဆင်သင့်ဖြစ်နေအောင် အောက်ပါ tools တွေကို ကြိုတင် install လုပ်ထားပါ -

  • Google Chrome
  • Visual Studio (VS) Code
  • Open in Browser (VS Code extension)
  • Git Bash (Windows သုံးသူများအတွက်)
  • Terminal (macOS သုံးသူများအတွက် - ၎င်းမှာ အရင်ကတည်းက ပါပြီးသားဖြစ်သည်)
  • Git နှင့် GitHub
  • Xcode (macOS သုံးသူများအတွက်သာ)
  • Homebrew (macOS သုံးသူများအတွက်သာ)
  • SSH keys

Quick Tips (အကြံပြုချက်များ)

  • Class repo ထဲက ဖိုင်တွေကို တိုက်ရိုက်မပြင်ပါနဲ့။ merge conflicts (ဖိုင်တွေ ထပ်နေတဲ့ပြဿနာ) မတက်အောင် ဖိုင်တွေကို တခြား folder တစ်ခုမှာ copy ကူးပြီးမှ စိတ်ကြိုက်ပြင်ဆင်ပါ။
  • မိမိရဲ့ username နဲ့ password တွေကို တစ်နေရာမှာ သေချာမှတ်ထားပါ။ GitLab လိုမျိုး နေရာတွေမှာ password ပြန်ပြောင်းရတာ အချိန်အတော်ကြာတတ်ပါတယ်။
